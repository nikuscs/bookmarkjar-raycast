# Bookmarkjar

Save bookmarks to your [Bookmarkjar](https://bookmarkjar.com) collection directly from Raycast.

## Setup

### Step 1: Create an API Key

**[Click here to create your API Key](https://bookmarkjar.com/dashboard/user/profile)**

Or run the "Bookmarkjar Create API Key" command in Raycast.

### Step 2: Configure the Extension

Run any Bookmarkjar command and paste your API key when prompted.

---

## Commands

- **Add Bookmark** - Paste one or more URLs to save
- **Add Bookmark from Clipboard** - Instantly save the URL in your clipboard
- **Bookmarkjar Create API Key** - Opens Bookmarkjar to create your API key
